== Description ==

Fully adjustable scroll tool for your site.
Easily adapts to any theme. You can change the size, shape, color, and much more. Dynamicly loads to the page and, thus does not affect on SEO. This plugin uses Color Picker - jQuery plugin for easy way to change colors of Your button. Tested in browsers: Internet Explorer 7 - 10 and last versions of Mozilla Firefox, Google Chrome and Opera.

Features:

* type of scroll (button or sidebar);
* fully customizable scroll tool;
* disable for small mobile screens (option);
* animated scroll motion (option);
* scroll down to last view position (option).
* change colors of the button using Color Picker palette

== Installation ==

1. Go to your WordPress admin panel and select "Plugins -> Add New" on your left sidebar menu.
2. Find and click on the "Upload" link.
3. Select a ZIP-file on your computer and click "Install Now" button.
4. Click on the "Activate Plugin" link when you receive "Plugin installed successfully." message.
5. Select "Appearance -> Scroll tool" on the left sidebar menu to adjust your scroll tool.
